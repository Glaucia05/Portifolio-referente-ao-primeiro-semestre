"Este é meu portfólio referente ao primeiro período do curso de Bacharelado em Ciências da Computação na instituição Pontifícia Universidade Católica do Paraná."

Nesse primeiro período foram apresentadas cinco matérias introdutórias a ciência da computação:

Raciocínio algoritmo
Resolução de problemas com logica matemática
Fundamentos de sistemas Ciberfísicos
Experiencia criativa
Filosofia

Cada matéria teve sua devida importância no meu processo de aprendizagem nesse primeiro periodo sendo assim, abaixo listei alguns detalhes sobre elas e como essas máterias se encaixam profissionalmente.

Raciocínio algoritmo

Nesse primeiro semestre, houve a introdução ao raciocínio algoritmo através da linguagem Python. Essa linguagem de programação de sintaxe simples e intuitiva que, com certeza é um bom ponto de partida para início da aprendizagem das linguagens. O raciocínio algoritmo, profissionalmente, nos ajuda a resolver problemas com programas implementados, ele nos induz a ter um pensamento computacional, estratégico, padronizado e automatizado, que é de extrema importância na área da tecnologia e suas variantes.
Em relação ao Python, sua importância no mercado de trabalho não é irrelevante. Sua linguagem tem forte potencial na resolução de problemas empresariais, como automatizar tarefas repetitivas como processar arquivos, gerar relatórios, fazer análises, varrer dados em massa (como planilhas, banco de dados, arquivos XML/JSON), lida com tabela de dados, base para sistemas, suporte para algoritmos ou visualização de dados.

Experiencia criativa

Nessa matéria, o foco foi totalmente a pratica com projetos maiores e geralmente em grupos, sendo assim, um tipo de experiencia introdutória ao que realmente vivemos no mercado de trabalho. Foi introduzido a linguagem Java, no ambiente de desenvolvimento Processinho, como também foi apresentado HTML, CSS e Java Script, no ambiente Visual Studio Code.
Em relação a linguagem Java, seu uso é estendido no mercado de trabalho, sendo usado em sistemas corporativos e bancos, em aplicações web, softwares embarcados e IoT e Back-end por exemplo, além de que é uma linguagem robusta, segura e orientada a objetos.
Já o Processing permite criar artes visuais com programação. Nesse período criamos até um aplicativo com essa linguagem.
Quanto ao HTML, CSS e Java Script, foi introduzido o uso dessas linguagens que são usadas para criar Web Sites como por exemplo sites institucionais, lojas online, blogs, etc. Sendo assim foram criados dois projetos principais com essas linguagens.

Resolução de problemas com logica matemática

Nessa matéria foi introduzido basicamente: a lógica matemática, tabela-verdade, manipulação sintática e regras de inferência, conceitos esses que são de extrema importância na área da tecnologia, pois, eles estruturam raciocínios, são usados em lógica de programação, em sistemas especialistas, regras de negócios, raciocínio automático, auditorias automatizadas e participando ativamente na criação de IAs.

Fundamentos de sistemas Ciberfísicos

Nessa matéria estudamos ativamente sobre fundamentos da computação e arquitetura de computadores principalmente. Abordamos temas como arquitetura de Von Neumann, tipos de memoria, características para um bom funcionamento de um computador e também introduzimos a programação do microcontrolador esp32, por Micropython.
