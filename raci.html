Projeto Máquina de Bebidas

Esse projeto consiste em construir uma máquina de Bebidas com as seguintes características: Ter ID, produtos, quantidade, valores e estoque específicos (que foi especificado no documento). A máquina deve ficar em loop aguardando o cliente escolher o código da bebida desejada. Assim que o cliente escolher o código da bebida, o programa deve validar se o código é válido e se há estoque disponível. Caso a escolha de bebida seja válida, o programa deve informar ao cliente o valor da bebida e o cliente deve inserir o valor pago. O valor pago deve ser maior ou igual ao valor do produto, caso não seja, o programa deve solicitar um novo valor. O programa deve calcular o troco especificando as notas e moedas que deverão ser devolvidas, tentando entregar o menor volume possível de notas e moedas Após a devolução do troco, o programa deve deduzir o estoque do respectivo produto comprado.
